# Python 网络编程学习仓库

这是一个用于学习和实践 Python 网络编程的仓库，包含多种网络爬虫和自动化工具的实现。

## 目录结构

- `bg_remove/`: 背景移除项目
- `db/`: 数据库相关代码
- `requests_html/`: 使用 `requests-html` 库的示例
- `requests/`: 使用 `requests` 库的示例
- `selenium/`: 使用 `selenium` 库的示例
- `src/`: 核心源代码

## 安装与使用

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-repo/learning_spider.git
   ```
2. 安装依赖：
   ```bash
   uv sync
   ```
